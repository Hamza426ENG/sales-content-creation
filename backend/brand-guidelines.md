# Edge Brand Guidelines — Sales Content Generation

- **Mission:** To connect the dots between impactful jobs and great global hires
- **Value Prop:** Hire World-Class Talent, Fully Managed
- **Website:** onedge.co
- **Headquarters:** 800 El Camino Real, Suite 180, Mountain View, CA 94040
- **Founded:** 2022
- **Platform Type:** Managed Talent Platform (NOT a staffing agency, NOT a BPO)

## Brand Voice & Tone
- **Archetype:** The Trusted Coach — patient, supportive, warm, direct, encouraging, clear
- **Personality Traits:** Visionary, Focused, Assured, Galvanizing
- Voice Principles:
  1. Embrace Candid Optimism — grounded but exciting and real
  2. Aim for Clarity, not Simplicity — straightforward, practical communicators
  3. Speak with Quiet Confidence — steady and unshakeable, not loud and brash
  4. Look to Encourage, not Cheerlead — supportive passion like a coach or mentor
- Use clear, direct language — avoid jargon
- Lead with outcomes and data
- Use "we" to refer to Edge
- Never use diminishing language like "resources" for people — use "professionals" or "talent"
- Never describe Edge talent as "the cheaper option" — focus on value

## Key Terminology
- Talent (not Associates) — for Edge professionals
- Customers (not Clients/Employers) — for businesses that hire through Edge
- Edge Edu (not Open Academy) — training platform
- Talent Network (not Marketplace) — talent pool
- Ecosystem (not Edge Services) — collection of platforms/products
- Edge Team (not Edge Stars) — internal employees

## Document Structure Guidelines
- Start with a compelling headline using DM Sans font
- Use section labels (pill badges with uppercase text, letter-spacing)
- Group related items in card-style sections with 1.5px borders and rounded corners
- Include stats/metrics sections with bold accent-colored numbers
- Include client testimonials in callout boxes
- End with a clear call-to-action
- Use dotted line dividers between major sections where appropriate

## Visual Style for Generated Documents
- **Primary Color:** Plum Purple (#4A0F70) — logo, headings, primary accents
- **Web Accent:** Violet (#7C3AED) — CTAs, links, interactive elements
- **Secondary:** Emerald Green (#059669) — positive metrics, success indicators
- **Gold/Amber:** (#D97706) — stats, key numbers, highlights
- **Dark sections:** (#0B0F1A) — CTA sections, closing sections
- **Light backgrounds:** (#FAFAFA) — alternating sections
- **Card backgrounds:** White with 1.5px #F0F0F4 borders, 20px border radius
- **Font:** DM Sans (headings bold/800, body regular/400)
- **Section labels:** Pill-shaped, 100px border-radius, uppercase, letter-spacing 1.8px
- Clean, modern design with generous whitespace
- Cards with subtle shadows and hover states
